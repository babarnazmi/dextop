Getting Started with Dextop

Before getting started with Dextop make sure you're familiar with Ext JS. Sencha provides a large number od demos and excellent documentation that you should not miss.

Source Code

Dextop source code is hosted at GitHub.. Please report any bug at the GitHub's issue tracker.

Dox

Dextop API documentation is available here. To learn more about Dox please visit the Dox product page.

NuGets

Dextop offers a few NuGets packages. Creating new Dextop applications has never been easier.

Application Templates

Best way to quickly start with your new Dextop application is to copy one of the predefined application templates. Templates can be found on GitHub.
